# ft_printf

## Description

ft_printf is one of the projects in Milestone 1 of the 42 curriculum.\
\
The goal of this project is to recreate the behavior of the standard C printf function. It allows formatted output to be written to the standard output and displays the values of different types of arguments using format specifiers.

## Instructions

First, clone the project repository using the following commands in your terminal:

```bash
git clone git@github.com:Peligorri/ft_printf.git ft_printf
cd ft_printf
```

Once the repository has been cloned, you can uncomment the main function and compile the program with:

```bash
make
cc -Wall -Werror -Wextra ft_printf.c libftprintf.a
```

Finally, you can run the program with:

```bash
./a.out
```
